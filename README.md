# TP3 SPRING

Ce projet est une application web complète comprenant l’authentification des utilisateurs, des tableaux de bord séparés pour les administrateurs et les utilisateurs, la gestion des produits ainsi qu’un système sécurisé de gestion des sessions. Il illustre une architecture claire et un contrôle d’accès structuré.
